CORONAVIRUS PUBLIC API - V1

About

A source of data on the growth of COVID-19 worldwide, using the method of "web scraping", the API manages to return updated data on the whole world and by countries every minute.

The project is completely open-source and open to contributions.

If you are interested in contributing to the project or have a problem using it, please contribute or report your problem in our GitHub repository.

Documentation

Available requests

Base URL: https://covid19-public-api.herokuapp.com/api/v1/

GET /status

Returns data from all over the world and from all countries.
Response example:

GET /status/{countryName}

Returns the data of the indicated country, if the country has a compound name, it must be passed without any spaces, example:
Bosnia and Herzegovina = BosniaandHerzegovina

Country names must always be given in English.
